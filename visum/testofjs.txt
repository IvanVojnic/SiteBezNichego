ВЗАИМОДЕЙТСВИЕ JAVASCRIPT C HTML:
прописываем тег в штмле, указываем его айди: <div id="myDiv">Salam</div>

в js обращаемся к этому элементу по айди:
создаем переменную. document - глобальный объект, у него есть много методов.
один из них - обращение по айди к тегу - getElementById("...");

var name = document.getElementById("MyDiv");

в переменную name мы кладем содержимое тега, теперь его нужно считать.
Создадим отдельную переменную и считаем в неее с помощью встроенной функции

var some = name.innerHTML;

теперь через переменную some можно взаимодействовать с содержимым тега
например: some = some + " текст" // добавили слово текст к строке, которая лежала в диве

взаимодействуем со содержимым , А НЕ ТЕГОМ! Чтобы изменить тег, нужно обратиться к
name.innerHTML, например:
name.innerHTML = "здесь был какой-то контент"; это строкой мы заменили
содержимое дива на строку: здесь был какой-то контент
